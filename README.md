# YMP TV Screensaver (Site)

![](https://ymp-co.github.io/YMP_TV_Screensaver/assets/images/play-store-512-337x337.png)

**📺 YMP TV Screensaver** is an application that allows users to use unusual screensavers and images on their devices.

The app comes pre-installed with more than 5 active screensavers and images that you can use on your desktop. In addition, screensavers can display additional widgets: weather widget, clock widget and others.

The application provides users with the ability to quickly and easily install and modify screensavers. 

**❗ This repository stores the project website and some web resources of the application.**
